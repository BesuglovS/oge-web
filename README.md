# ОГЭ - Интерактивные задания по информатике

Интерактивное веб-приложение для подготовки к Основному государственному экзамену (ОГЭ) по информатике. Проект содержит задания с различными типами интерактива, поддерживающими как мышь, так и сенсорные экраны.

## 🎯 Задания

### Задание 1: Информационный объём

Определение информационного объёма сообщения. Задачи на кодирование текста, вычисление объёма при удалении и добавлении слов.

**Типы задач:**

- Вычисление информационного объёма текстового сообщения
- Определение изменения объёма при редактировании текста
- Расчёт количества символов по известному объёму

### Задание 2: Декодирование сообщений

Расшифровка закодированного сообщения с использованием таблицы кодов. Задачи на применение условия Фано.

**Типы задач:**

- Однозначное декодирование сообщения
- Проверка условия Фано
- Построение кодовых цепочек

### Задание 3: Истинность высказывания

Определение истинности составного высказывания. Задачи на логические операции И, ИЛИ, НЕ.

**Типы задач:**

- Вычисление значения логического выражения
- Анализ составных высказываний
- Работа с таблицами истинности

### Задание 4: Анализ модели объектов

Определение кратчайшего пути между пунктами. Задачи на анализ графа дорог и поиск оптимального маршрута.

**Типы задач:**

- Поиск кратчайшего пути в графе
- Анализ табличной модели дорог
- Определение оптимального маршрута

### Задание 5: Анализ алгоритмов для исполнителя

Составьте алгоритм для исполнителя или определите неизвестный параметр. Задачи на анализ простых алгоритмов с фиксированным набором команд.

**Типы задач:**

- Определение результата работы алгоритма
- Поиск входных данных по результату
- Анализ программ для исполнителей

### Задание 6: Анализ программы

Проанализируйте программу на языке программирования. Задачи на определение количества запусков с «YES» или «NO» или поиск параметра A.

**Типы задач:**

- Определение количества запусков программы
- Поиск параметра по условию
- Анализ условных операторов

### Задание 7: Составление URL-адреса

Составьте URL-адрес по заданным параметрам: протокол, домен, путь и имя файла.

**Типы задач:**

- Формирование URL из компонентов
- Анализ структуры веб-адресов
- Работа с протоколами и доменами

### Задание 8: Множества и круги Эйлера

Решите задачи на поисковые запросы с использованием кругов Эйлера. Операции И, ИЛИ, НЕ над множествами.

**Типы задач:**

- Вычисление количества страниц по запросам
- Анализ логических связок в поиске
- Построение диаграмм Эйлера-Венна

### Задание 10: Системы счисления

Перевод чисел между системами счисления, арифметические операции, сравнение чисел, поиск по условию.

**Типы задач:**

- Перевод чисел между системами счисления
- Арифметические операции в различных системах
- Сравнение и поиск чисел по условию

## 🛠️ Технологии

- **HTML5** — структура страниц
- **CSS3** — стилизация, градиенты, анимации, адаптивный дизайн
- **JavaScript** — интерактивность, drag-and-drop API, touch events
- **Yandex.Metrika** — аналитика посещений

## 🚀 Запуск

Просто откройте `oge.html` в браузере или запустите локальный сервер:

```bash
# С помощью Python
python -m http.server 8000

# С помощью Node.js
npx serve
```

Затем откройте http://localhost:8000 в браузере.

## ✨ Особенности

- **Адаптивный дизайн** — работает на десктопе и мобильных устройствах
- **Touch-поддержка** — перетаскивание пальцем на сенсорных экранах
- **Мгновенная проверка** — визуальная обратная связь при размещении элементов
- **Случайная генерация** — новые задания при каждом запуске
- **Подсказки** — помощь при затруднениях
- **Подробные результаты** — разбор ошибок после проверки
- **Теоретические материалы** — раздел с теорией для каждого задания

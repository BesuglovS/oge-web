📚 Теория: Задание 11 ОГЭ — Информационный поиск

💡 Что такое информационный поиск?

Информационный поиск — это процесс нахождения нужной информации в одном или нескольких файлах. На экзамене вам предоставляется архив с файлами, которые нужно распаковать и открыть в текстовом редакторе или браузере, чтобы найти ответ на поставленный вопрос.

В задании 11 ОГЭ нужно уметь находить информацию в текстовых файлах (форматов .txt, .htm, .html, .doc, .docx, .rtf и других), используя поисковые средства операционной системы и текстового редактора.

📁 Структура файлов в задании

Архив с данными содержит тексты литературных произведений, организованные в папки. Внутри архива обычно есть каталоги Проза и Стихи, а внутри них — подкаталоги с фамилиями авторов.

📄 Какие бывают типы вопросов

🔎 Поиск по контексту

Вам даётся описание ситуации из произведения. Нужно найти фрагмент текста, содержащий описание, и извлечь из него требуемую информацию (город, предмет, имя, название и т.п.).

📖 Поиск по автору и произведению

Указан автор и название произведения. Нужно найти в тексте ответ на вопрос: название города, предмета, персонажа и т.д.

🏷️ Поиск по ключевым словам

Даны ключевые слова или фразы, которые встречаются в тексте. Нужно найти файл с этими словами и извлечь из него ответ.

🔍 Алгоритм решения

  1. Распакуйте архив с файлами в удобную папку на компьютере.
  2. Внимательно прочитайте вопрос и выделите ключевые слова для поиска (название произведения, автора, персонажей, предметов, мест).
  3. Определите нужную папку (например, Проза/Гончаров или Стихи/Есенин).
  4. Откройте нужный файл в текстовом редакторе или браузере.
  5. Используйте поиск по тексту (обычно Ctrl+F), чтобы быстро найти ключевые слова.
  6. Прочитайте контекст вокруг найденного фрагмента и извлеките требуемую информацию.
  7. Запишите ответ в поле ввода.

💡 Совет

Если в одном файле находятся несколько произведений одного автора, внимательно проверяйте заголовки — они помогут определить, где начинается нужное произведение.

🔎 Использование поиска (Ctrl+F)

Ctrl+F (или Cmd+F на Mac) — это универсальная команда поиска по тексту, которая работает во всех текстовых редакторах и браузерах.

📝 Как использовать поиск

1. Откройте файл в текстовом редакторе (Блокнот, Word, Notepad++) или браузере.
2. Нажмите Ctrl+F — появится строка поиска.
3. Введите ключевое слово или фразу из вопроса.
4. Нажимайте Enter или кнопку «Найти далее», чтобы переходить между совпадениями.
5. Прочитайте предложение вокруг найденного слова, чтобы понять контекст.

💡 Какие слова искать?

Выделите из вопроса самые уникальные слова: имена персонажей, названия предметов, географические названия, необычные словосочетания. Избегайте слишком коротких или распространённых слов (например, «и», «в», «он»).

📂 Какие форматы файлов встречаются

  • .txt — простой текстовый файл (открывается Блокнотом)
  • .htm, .html — веб-страница (открывается браузером)
  • .doc, .docx — документ Microsoft Word
  • .rtf — формат Rich Text Format (открывается Word или Блокнотом)
  • .odt — документ OpenDocument (открывается LibreOffice Writer)
  • .pdf — документ в формате PDF

⚠️ Важно!

Файлы могут иметь разные форматы. Если файл не открывается в одном редакторе, попробуйте открыть его в другом. Текстовые файлы .txt можно открыть любым текстовым редактором, а .htm — браузером.

✏️ Пример решения

Пример задачи

«В романе И. А. Гончарова „Обыкновенная история", текст которого приведён в подкаталоге каталога Проза, один из героев называет город, из которого ему привозят солёные огурцы. С помощью поисковых средств операционной системы и текстового редактора или браузера выясните название этого города.»

Решение:
1. Распаковываем архив, находим папку Проза/Гончаров.
2. Открываем файл с романом «Обыкновенная история» в браузере или текстовом редакторе.
3. Нажимаем Ctrl+F и вводим ключевые слова: «огурцы» или «солёные».
4. Находим фрагмент: «…из Москвы мне привозят солёные огурцы».
5. Ответ: Москва.

Пример задачи

«В романе Ф. М. Достоевского „Идиот", текст которого приведён в подкаталоге каталога Проза, один из героев высказывает мнение о том, что русская литература вся нерусская, за исключением произведений трёх авторов, двое из которых Пушкин и Ломоносов. С помощью поисковых средств ... выясните фамилию третьего автора.»

Решение:
1. Открываем папку Проза/Достоевский.
2. Открываем файл с романом «Идиот».
3. Нажимаем Ctrl+F и ищем «Пушкин» или «Ломоносов».
4. Читаем контекст: «...вся литература наша нерусская, кроме Пушкина, Ломоносова и Державина».
5. Ответ: Державин.

⚠️ Частые ошибки

  • Искать не в том файле — внимательно читайте, в каком подкаталоге и у какого автора нужно искать
  • Искать слишком общие слова — используйте уникальные ключевые слова, а не «город», «человек», «сказал»
  • Не прочитать контекст — найденное слово может быть не в том значении; читайте предложение целиком
  • Неверно записать ответ — ответом может быть город, имя, фамилия, название; следите за падежом и формой

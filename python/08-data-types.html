08. Типы данных

int, float, str, bool, NoneType, type(), isinstance()

Основные встроенные типы

В Python у каждого значения есть тип:


# Числовые типы
x = 10          # int — целое число
y = 3.14        # float — число с плавающей точкой
z = 1 + 2j      # complex — комплексное число

# Строковый тип
name = "Python"  # str — строка

# Логический тип
flag = True      # bool — булево значение

# Пустой тип
result = None    # NoneType — отсутствие значения

Определение типа — type() и isinstance()

Узнать тип переменной:


print(type(10))          # 
print(type("hello"))     # 
print(type(True))        # 

# isinstance — проверка принадлежности типу
print(isinstance(10, int))       # True
print(isinstance("abc", str))    # True
print(isinstance(10, (int, float)))  # True

Преобразование типов

Явное преобразование между типами:


# В целое число
print(int(3.14))       # 3
print(int("42"))       # 42

# В число с плавающей точкой
print(float(5))        # 5.0
print(float("2.5"))    # 2.5

# В строку
print(str(100))        # "100"
print(str(3.14))       # "3.14"

# В булево значение
print(bool(1))         # True
print(bool(0))         # False
print(bool(""))        # False
print(bool("text"))    # True

Изменяемые и неизменяемые типы

Неизменяемые (immutable): int, float, str, bool, tuple, frozenset

Изменяемые (mutable): list, dict, set


# Строки неизменяемы
s = "hello"
# s[0] = "H"  # Ошибка! Нельзя изменить символ
s = "H" + s[1:]  # Нужно создать новую строку

# Списки изменяемы
lst = [1, 2, 3]
lst[0] = 10  # Работает!

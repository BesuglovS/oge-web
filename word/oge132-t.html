📚 Теория: Задание 13.2 ОГЭ — Создание текстового документа

🎯 Что такое задание 13.2?

Задание 13.2 — практическое задание ОГЭ по информатике, в котором требуется создать текстовый документ в текстовом редакторе (Microsoft Word или LibreOffice Writer), точно воспроизведя оформление текста по образцу.

Участнику экзамена предоставляется каталог с файлами, содержащий текстовый файл (*.txt) с текстом задания — в нём описан текст, который нужно набрать, и требования к его оформлению. В отличие от задания 13.1 (презентация), задание 13.2 менее творческое, но требует точного соответствия результата работы образцу, представленному в условии.

⚠️ Важно!

Документ необходимо сохранить в формате *.odt (OpenDocument Text). Имя файла сообщат организаторы экзамена. Текстовый редактор на экзамене — Microsoft Word или LibreOffice Writer (в зависимости от региона).

📝 Пример задания

Пример: «Варенье из ежевики»

Создайте в текстовом редакторе документ и напишите в нём следующий текст, точно воспроизведя всё оформление текста, имеющееся в образце.

ВАРЕНЬЕ ИЗ ЕЖЕВИКИ
Перебрать килограмм ежевики, удалить мятые ягоды и веточки. Высыпать плоды на дуршлаг, помыть и дать стечь воде. Засыпать сахарным песком, оставить на 4 часа. Поставить сахарно-плодовую смесь на плиту. Постоянно помешивая, довести до кипения и проварить 3 минуты. Дать остыть. Повторить процедуру 3 раза. В конце по вкусу добавить лимонную кислоту, разложить горячее ежевичное варенье по стерилизованным банкам, закатать банки.

Требования к оформлению: шрифт 14 пт, отступ первой строки 1 см, выравнивание по ширине, междустрочный интервал от одинарного до полуторного. Заголовок — по центру. Таблица — необходимое количество строк и столбцов, выравнивание по образцу.

📐 Структура документа

В задании 13.2 документ состоит из двух основных объектов, которые необходимо оформить в точном соответствии с образцом:

📄 Основной текст

  • Один или два абзаца (до ~200 символов)
  • Шрифт 14 пт, обычное начертание
  • Отступ первой строки 1 см
  • Междустрочный интервал от одинарного до полуторного
  • Выравнивание по ширине
  • Выделение слов (полужирный/курсив/подчёркивание)

📊 Таблица

  • Необходимое количество строк и столбцов
  • Заголовок таблицы — полужирный, по центру
  • Второй столбец — по центру
  • Первый столбец (кроме заголовка) — по левому краю
  • Ширина таблицы меньше ширины текста
  • Таблица по центру страницы

📋 Детальная структура

Заголовок

  • Располагается перед таблицей
  • Выравнивание по центру
  • Размер шрифта — 14 пт (может быть полужирным по образцу)

Таблица

  • Располагается между заголовком и основным текстом
  • Количество строк и столбцов — по образцу
  • Заголовок таблицы (первая строка) — полужирное начертание, выравнивание по центру
  • Первый столбец (кроме заголовка) — выравнивание по левому краю
  • Второй столбец — выравнивание по центру
  • Ширина таблицы — меньше ширины основного текста
  • Таблица выровнена по центру страницы

Основной текст

  • Располагается после таблицы
  • Шрифт 14 пт, обычное начертание
  • Выравнивание по ширине
  • Отступ первой строки 1 см
  • Междустрочный интервал — от одинарного до полуторного
  • При необходимости — выделение слов полужирным, курсивом или подчёркиванием по образцу

📄 Требования к тексту

  • Текст в абзаце выровнен по ширине (за исключением заголовка)
  • Правильно установлен абзацный отступ (1 см), не допускается использование пробелов
  • Разбиение текста на строки осуществляется текстовым редактором автоматически (не используются разрывы строк)
  • Допускается не более 5 ошибок (орфографических, пунктуационных, пробелы, знаки, пропуски) в основном тексте
  • Допускается не более 3 ошибок (орфографических, пунктуационных, пробелы, знаки, пропуски) в таблице

📋 Критерии оценивания

Баллы Описание
2 балла Задание выполнено правильно. Контролируется выполнение всех элементов оформления: размер шрифта 14 пт, верное выделение слов, междустрочный интервал, абзацный отступ 1 см, выравнивание по ширине, правильная структура таблицы, интервал между текстом и таблицей без использования пустого абзаца.
1 балл В работе допущена суммарно не более одной ошибки в структуре документа, или допущена одна ошибка одного из следующих типов: неверный размер шрифта, неверно задан абзацный отступ, ошибка в выравнивании, нарушение межстрочного интервала, ошибка в оформлении таблицы (пропуск строки/столбца).
0 баллов Работа не соответствует требованиям, отсутствует таблица, либо допущено более одной грубой ошибки.

Блоки проверки

📄 Основной текст

Размер шрифта 14 пт, верное выделение слов, междустрочный интервал, абзацный отступ 1 см, выравнивание по ширине, интервал между текстом и таблицей без пустых абзацев, не более 5 орфографических ошибок.

📊 Таблица

Необходимое количество строк и столбцов, полужирный заголовок, правильное выравнивание столбцов, ширина меньше ширины текста, таблица по центру, не более 3 орфографических ошибок.

🎨 Оформление

Интервал между объектами 14–24 пт (без пустых абзацев), отступ первой строки 1 см (не пробелами), отсутствие разрывов строк внутри абзаца, формат файла *.odt.

📌 Важное замечание

При проверке контролируются только те элементы, которые явно перечислены в критериях. Расхождение работы с образцом в параметрах, не указанных в критериях, не влияет на оценку.

Исключение: если параметры шрифтового или абзацного оформления искажены столь значительно, что явно нарушается эстетическое восприятие текста, оценка снижается на балл.

📌 Порядок выполнения задания

  1. Откройте текстовый редактор (Microsoft Word или LibreOffice Writer)
  2. Прочитайте задание в предоставленном текстовом файле (*.txt) — обратите внимание на образец оформления и требования
  3. Наберите заголовок — выровняйте по центру, установите размер шрифта 14 пт. Выделите нужные слова полужирным или курсивом по образцу
  4. Создайте таблицу: Вставка → Таблица → укажите количество строк и столбцов по образцу. Заполните ячейки, настройте выравнивание (заголовок и второй столбец — по центру, первый столбец — по левому краю)
  5. Установите ширину таблицы — меньше ширины основного текста (через Свойства таблицы). Выровняйте таблицу по центру страницы
  6. Наберите основной текст:
    • Шрифт 14 пт, обычное начертание
    • Выравнивание по ширине
    • Отступ первой строки 1 см (Формат → Абзац → Первая строка → Отступ → 1 см)
    • Междустрочный интервал — от одинарного до полуторного
  7. Установите интервал между заголовком/текстом и таблицей — от 14 до 24 пт (через настройки абзаца, НЕ пустым абзацем!)
  8. Выделите слова в тексте и таблице полужирным, курсивом или подчёркиванием в соответствии с образцом
  9. Проверьте невидимые символы: включите режим отображения невидимых символов (¶) — убедитесь, что нет лишних пробелов для отступа, нет пустых абзацев
  10. Сохраните файл в формате *.odt (Файл → Сохранить как → Тип файла → ODF Text Document (.odt))

⚠️ Типичные ошибки

❌ Использование пробелов для абзацного отступа

Ошибка: вместо настройки отступа первой строки через диалог «Абзац» ученик ставит несколько пробелов в начале абзаца.

Решение: используйте Формат → Абзац → Отступы → Первая строка → Отступ → 1 см. Включите показ невидимых символов (¶), чтобы убедиться в отсутствии лишних пробелов и табуляции.

❌ Пустые абзацы для интервала между текстом и таблицей

Ошибка: несколько нажатий Enter между текстом и таблицей вместо настройки интервала через параметры абзаца.

Решение: установите интервал после текстового абзаца (или перед таблицей) через Формат → Абзац → Интервал после/перед.

❌ Неверный размер шрифта

Ошибка: использование шрифта 12 пт (по умолчанию) вместо 14 пт, указанного в задании.

Решение: перед началом работы установите размер шрифта 14 пт на вкладке «Главная». Проверьте, что размер применён ко всему тексту.

❌ Неверная ширина таблицы

Ошибка: таблица занимает всю ширину страницы или, наоборот, слишком узкая.

Решение: ширина таблицы должна быть меньше ширины основного текста. Выделите таблицу → Свойства таблицы → укажите ширину (например, 80–90% от ширины текста).

❌ Неверное выравнивание в таблице

Ошибка: все ячейки таблицы выровнены одинаково (например, только по центру или только по левому краю).

Решение: заголовок и второй столбец — по центру, первый столбец (кроме заголовка) — по левому краю.

❌ Использование разрывов строк внутри абзаца

Ошибка: принудительный перенос строки (Shift + Enter) внутри абзаца вместо того, чтобы позволить редактору автоматически разбивать текст на строки.

Решение: текст должен переноситься автоматически. Включите невидимые символы и убедитесь, что внутри абзаца нет символов разрыва строки (↵).

❌ Сохранение в неверном формате

Ошибка: сохранение файла в формате .docx или .doc вместо .odt.

Решение: при сохранении выберите тип файла «OpenDocument Text (*.odt)». В Word: Файл → Сохранить как → Тип файла → OpenDocument Text.

💡 Полезные советы

📌 Как получить максимальный балл

  • Используйте настройки, а не пробелы. Абзацный отступ — через диалог «Абзац». Интервал между текстом и таблицей — тоже через «Абзац» (интервал после/перед).
  • Включите показ невидимых символов (¶) — так вы увидите лишние пробелы, пустые абзацы, разрывы строк и сможете их удалить.
  • Проверьте ширину таблицы — она должна быть меньше ширины основного текста.
  • Проверьте выравнивание в таблице — для каждого столбца своё выравнивание (заголовок и второй — по центру, первый — по левому краю).
  • Не используйте разрывы строк для перехода на новую строку внутри абзаца — разбиение на строки должно выполняться автоматически.
  • Сохраните в правильном формате — *.odt (не .docx, не .doc).
  • Проверьте орфографию — ошибки в тексте снижают балл (допускается не более 5 ошибок в тексте и 3 в таблице).

🔑 Ключевые сочетания клавиш

  • Ctrl + J — выравнивание по ширине
  • Ctrl + E — выравнивание по центру
  • Ctrl + S — сохранить документ
  • Ctrl + Z — отменить последнее действие
  • Ctrl + B — полужирное начертание
  • Ctrl + I — курсив
  • Ctrl + U — подчёркивание
  • Ctrl + Shift + 8 (Word) — показать невидимые символы
  • Ctrl + F10 (Writer) — показать невидимые символы

📋 Примеры из практики оценивания

✅ Пример работы на 2 балла

Основной текст: размер шрифта 14 пт, отступ первой строки 1 см, выравнивание по ширине, все слова выделены правильно, междустрочный интервал одинарный, интервал между текстом и таблицей установлен через настройки абзаца (16 пт).

Таблица: верное количество строк и столбцов, заголовок полужирный по центру, второй столбец по центру, первый столбец по левому краю, ширина таблицы меньше ширины текста, таблица по центру страницы.

Орфография: опечаток нет. Оценка — 2 балла.

⚠️ Пример работы на 1 балл (одна ошибка)

Пропущена одна строка таблицы (отсутствует последняя строка с данными). Все остальные элементы выполнены верно. Ошибка в структуре таблицы — 1 балл.

⚠️ Пример работы на 1 балл (ошибка оформления)

Размер шрифта основного текста указан 12 пт вместо 14 пт. Все остальные требования выполнены верно. Одна ошибка — 1 балл.

❌ Пример работы на 0 баллов

Абзацный отступ выполнен пробелами, интервал между текстом и таблицей — пустыми абзацами (два нажатия Enter), таблица отсутствует. Множественные нарушения — 0 баллов.

📌 Интересный случай из практики

Участник экзамена полностью выполнил работу в области верхнего колонтитула. Проверка показала: размер шрифта, отступ первой строки и выравнивание абзаца выполнены верно, все слова правильно выделены, опечаток нет. Таблица имеет структуру, полностью соответствующую образцу.

Результат: согласно критериям, работу необходимо оценить максимальным баллом — 2 балла. Единственное отличие — размещение текста и таблицы в области колонтитула, что не является ошибкой по критериям.

⚠️ Однако при подготовке к экзамену не следует поощрять учеников выполнять задание подобным образом.

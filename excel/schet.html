🔢 Функция СЧЁТ в Excel

💡 Что делает функция СЧЁТ?

Функция СЧЁТ — это статистическая функция, которая подсчитывает количество ячеек, содержащих числа, в указанном диапазоне.

Функция СЧЁТ игнорирует текстовые значения, логические значения (ИСТИНА/ЛОЖЬ), пустые ячейки и ячейки с ошибками. Она подсчитывает только ячейки, содержащие числовые значения.

📝 Синтаксис функции СЧЁТ

=СЧЁТ(значение1; [значение2]; ...)
значение1 — обязательный аргумент, первое значение или диапазон для подсчёта
[значение2]; ... — необязательные аргументы, дополнительные значения или диапазоны

🔍 Аргументы функции

Тип аргумента Пример Результат
Отдельные числа =СЧЁТ(10; 20; 30) 3 (все три аргумента — числа)
Диапазон ячеек =СЧЁТ(A1:A10) Количество ячеек с числами в A1-A10
Несколько диапазонов =СЧЁТ(A1:A5; C1:C5) Количество чисел в A1-A5 и C1-C5
Смешанные аргументы =СЧЁТ(A1:A3; "текст"; 100) Количество чисел в A1-A3 плюс число 100

🧮 Примеры использования

Пример 1: Подсчёт оценок

Условие: В ячейках A1:A10 находятся оценки учеников (некоторые ячейки пустые). Нужно подсчитать, сколько оценок выставлено.

Данные:
A1=4, A2=5, A3=пусто, A4=3, A5=4, A6=5, A7=пусто, A8=4, A9=5, A10=пусто

Формула: =СЧЁТ(A1:A10)

Результат: 7 (пустые ячейки игнорируются)

Пример 2: Подсчёт числовых данных

Условие: В ячейках B1:B8 находятся данные разного типа. Нужно подсчитать, сколько из них числовых.

Данные:
B1=100, B2="нет", B3=200, B4=ИСТИНА, B5=300, B6=пусто, B7=400, B8="данные"

Формула: =СЧЁТ(B1:B8)

Результат: 4 (только B1, B3, B5, B7 — числа)

Пример 3: Подсчёт с несколькими диапазонами

Условие: В таблице есть данные за два месяца: январь (C1:C10) и февраль (D1:D10). Нужно подсчитать общее количество числовых записей.

Данные:
В январе 8 чисел из 10 ячеек, в феврале 7 чисел из 10 ячеек

Формула: =СЧЁТ(C1:C10; D1:D10)

Результат: 8 + 7 = 15

🔗 Особенности работы функции СЧЁТ

Игнорирование нечисловых значений

Пример: =СЧЁТ(A1:A4), где A1=10, A2="текст", A3=20, A4=пусто
Результат: 2 (только A1 и A3 — числа)

Подсчёт с нулями

Пример: =СЧЁТ(A1:A4), где A1=10, A2=0, A3=20, A4=0
Результат: 4 (все ячейки содержат числа, включая нули)

Игнорирование логических значений

Пример: =СЧЁТ(A1:A3), где A1=10, A2=ИСТИНА, A3=20
Результат: 2 (ИСТИНА игнорируется)

⚠️ Частые ошибки

  • Путаница с функцией СУММ — СЧЁТ подсчитывает количество чисел, а СУММ складывает числа
  • Ожидание подсчёта всех ячеек — функция СЧЁТ подсчитывает только числовые значения
  • Путаница с СЧЁТЗ — СЧЁТ подсчитывает только числа, а СЧЁТЗ подсчитывает все непустые ячейки
  • Забыть знак равенства — все формулы в Excel начинаются с =
  • Неправильные разделители — в русской версии Excel аргументы разделяются точкой с запятой (;), а не запятой (,)

🎯 Алгоритм работы с функцией СЧЁТ

  1. 📝 Определите, какие ячейки нужно подсчитать
  2. 🔢 Выберите диапазон ячеек или отдельные значения
  3. 🧮 Запишите формулу: =СЧЁТ(диапазон_или_значения)
  4. ✅ Проверьте, что в диапазоне есть числовые данные
  5. 🔍 Убедитесь, что текстовые значения не будут учтены
  6. 📊 Протестируйте формулу на тестовых данных

✏️ Практическое задание

Задача: В таблице Excel записаны результаты теста учеников (B2:B20). Некоторые ученики не сдали тест (ячейки пустые или содержат "не сдал"). Напишите формулу для ячейки B21, которая подсчитывает количество учеников, сдавших тест (получивших числовую оценку).

Решение:
Функция СЧЁТ автоматически игнорирует пустые ячейки и текстовые значения:
=СЧЁТ(B2:B20)

Проверка:
Если из 19 ячеек 12 содержат оценки (2, 3, 4, 5), 5 пустых, 2 содержат "не сдал"
Результат: 12

🔗 Связанные функции

СЧЁТЕСЛИ — подсчёт ячеек по одному условию
СЧЁТЕСЛИМН — подсчёт ячеек по нескольким условиям
СЧЁТЗ — подсчёт всех непустых ячеек
СУММ — суммирование чисел в диапазоне
СРЗНАЧ — вычисление среднего арифметического

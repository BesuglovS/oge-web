📊🔍🔍 Функция СРЗНАЧЕСЛИМН в Excel

💡 Что делает функция СРЗНАЧЕСЛИМН?

Функция СРЗНАЧЕСЛИМН — это статистическая функция, которая вычисляет среднее арифметическое значений ячеек, удовлетворяющих нескольким условиям одновременно.

Функция СРЗНАЧЕСЛИМН используется для анализа данных, когда нужно найти среднее значение элементов, соответствующих нескольким критериям одновременно.

📝 Синтаксис функции СРЗНАЧЕСЛИМН

=СРЗНАЧЕСЛИМН(диапазон_среднего; диапазон_условия1; условие1; [диапазон_условия2]; [условие2]; ...)
диапазон_среднего — диапазон значений для вычисления среднего
диапазон_условия1 — первый диапазон для проверки
условие1 — первый критерий для отбора
диапазон_условия2, условие2, ... — дополнительные пары диапазон-условие

🧮 Пример использования

Пример: Средняя зарплата сотрудников отдела "IT" с зарплатой больше 50000

Условие: В столбце A отделы, в столбце B зарплаты. Найти среднюю зарплату сотрудников отдела "IT" с зарплатой больше 50000.

Данные:
A1:A6 = {"IT", "HR", "IT", "Sales", "IT", "HR"}
B1:B6 = {50000, 40000, 55000, 45000, 60000, 42000}

Формула: =СРЗНАЧЕСЛИМН(B1:B6; A1:A6; "IT"; B1:B6; ">50000")

Расчёт:
Сотрудники отдела IT: A1, A3, A5
Их зарплаты: 50000, 55000, 60000
Зарплата > 50000: 55000, 60000
Среднее: (55000 + 60000) / 2 = 57500
Результат: 57500

🔗 Связанные функции

СРЗНАЧ — вычисляет среднее арифметическое
СРЗНАЧЕСЛИ — вычисляет среднее по одному условию
СУММЕСЛИМН — суммирует значения по нескольким условиям
СЧЁТЕСЛИМН — подсчитывает ячейки по нескольким условиям

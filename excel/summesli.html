💰 Функция СУММЕСЛИ в Excel

💡 Что делает функция СУММЕСЛИ?

Функция СУММЕСЛИ — это математическая функция, которая суммирует значения ячеек в диапазоне, удовлетворяющих заданному условию. Функция возвращает сумму только тех значений, которые соответствуют критерию.

Функция СУММЕСЛИ используется для анализа данных, когда нужно вычислить сумму значений, соответствующих определённому условию (например, сумма продаж по определённому товару, сумма зарплат сотрудников определённого отдела, сумма баллов учеников с оценкой "5" и т.д.).

📝 Синтаксис функции СУММЕСЛИ

=СУММЕСЛИ(диапазон_условия; условие; [диапазон_суммирования])
диапазон_условия — обязательный аргумент, диапазон для проверки условия
условие — обязательный аргумент, критерий для отбора ячеек
диапазон_суммирования — необязательный аргумент, диапазон значений для суммирования

🔍 Аргументы функции

Аргумент Описание Пример
диапазон_условия Диапазон ячеек, в которых проверяется условие A1:A10, B2:B100
условие Критерий для отбора ячеек (число, текст, выражение) 5, "яблоки", ">100"
диапазон_суммирования Диапазон значений для суммирования (если опущен, суммируется диапазон_условия) C1:C10, D2:D100

📊 Примеры использования

Пример 1: Сумма продаж определённого товара

Условие: В столбце A названия товаров, в столбце B суммы продаж. Нужно вычислить сумму продаж товара "Яблоки".

Данные:
A1:A5 = {"Яблоки", "Груши", "Яблоки", "Бананы", "Яблоки"}
B1:B5 = {100, 150, 200, 120, 180}

Формула: =СУММЕСЛИ(A1:A5; "Яблоки"; B1:B5)

Расчёт:
Находим "Яблоки" в A1, A3, A5
Суммируем соответствующие значения из B1, B3, B5: 100 + 200 + 180 = 480
Результат: 480

Пример 2: Сумма зарплат сотрудников отдела "IT"

Условие: В столбце A отделы, в столбце B зарплаты. Нужно вычислить сумму зарплат сотрудников отдела "IT".

Данные:
A1:A6 = {"IT", "HR", "IT", "Sales", "IT", "HR"}
B1:B6 = {50000, 40000, 55000, 45000, 60000, 42000}

Формула: =СУММЕСЛИ(A1:A6; "IT"; B1:B6)

Расчёт:
Находим "IT" в A1, A3, A5
Суммируем зарплаты: 50000 + 55000 + 60000 = 165000
Результат: 165000

Пример 3: Сумма чисел больше 50

Условие: В диапазоне C1:C10 находятся числа. Нужно вычислить сумму чисел, которые больше 50.

Данные: C1:C10 = {30, 60, 45, 80, 25, 90, 55, 40, 70, 35}

Формула: =СУММЕСЛИ(C1:C10; ">50")

Расчёт:
Числа > 50: 60, 80, 90, 55, 70
Сумма: 60 + 80 + 90 + 55 + 70 = 355
Результат: 355

🔗 Особенности работы функции СУММЕСЛИ

Диапазон суммирования

Правило: Если третий аргумент опущен, суммируются значения из диапазона_условия.
Пример: =СУММЕСЛИ(A1:A10; ">10") — суммирует значения из A1:A10, которые > 10

Символы подстановки

* — любое количество символов
? — один символ
Пример: =СУММЕСЛИ(A1:A10; "Яб*"; B1:B10) — товары, начинающиеся с "Яб"

Регистр букв

Правило: Функция СУММЕСЛИ не учитывает регистр букв.
Пример: =СУММЕСЛИ(A1:A10; "it"; B1:B10) найдёт и "IT", и "It", и "it"

⚠️ Частые ошибки

  • Неправильный порядок аргументов — =СУММЕСЛИ(условие; диапазон_условия; диапазон_суммирования) неправильно
  • Забывание кавычек для текста — =СУММЕСЛИ(A1:A10; Яблоки; B1:B10) неправильно
  • Несовпадающие размеры диапазонов — диапазон_условия и диапазон_суммирования должны быть одного размера
  • Использование запятой вместо точки с запятой — в русской версии Excel аргументы разделяются точкой с запятой (;)
  • Путаница с СУММ и СУММЕСЛИ — СУММ суммирует все значения, СУММЕСЛИ — только по условию

🎯 Алгоритм работы с функцией СУММЕСЛИ

  1. 📝 Определите диапазон для проверки условия
  2. 🔍 Сформулируйте условие для отбора ячеек
  3. 💰 Определите диапазон значений для суммирования (если нужен)
  4. 🧮 Запишите формулу: =СУММЕСЛИ(диапазон_условия; условие; [диапазон_суммирования])
  5. ✅ Проверьте правильность аргументов и кавычек
  6. 📊 Протестируйте формулу на примере данных

✏️ Практическое задание

Задача: В таблице Excel записаны данные о продажах: столбец A — названия товаров, столбец B — количество проданных единиц, столбец C — цена за единицу. Напишите формулу для ячейки D1, которая вычисляет общую выручку от продажи товара "Молоко".

Решение:
1. Диапазон для проверки условия: A1:A100 (названия товаров)
2. Условие: "Молоко"
3. Нужно вычислить: количество × цена для товара "Молоко"
4. Сначала находим строки с "Молоко", затем умножаем B×C для этих строк
5. Формула: =СУММЕСЛИ(A1:A100; "Молоко"; B1:B100 * C1:C100) — НЕПРАВИЛЬНО!
6. Правильное решение: нужно добавить вспомогательный столбец D = B×C, затем =СУММЕСЛИ(A1:A100; "Молоко"; D1:D100)
Или: =СУММПРОИЗВ((A1:A100="Молоко")*(B1:B100)*(C1:C100))

🔗 Связанные функции

СУММ — суммирует все значения в диапазоне
СУММЕСЛИМН — суммирует значения по нескольким условиям
СЧЁТЕСЛИ — подсчитывает количество ячеек по условию
СРЗНАЧЕСЛИ — вычисляет среднее значение по условию
СУММПРОИЗВ — умножает и суммирует массивы (для сложных условий)

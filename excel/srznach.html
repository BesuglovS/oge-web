📊 Функция СРЗНАЧ в Excel

💡 Что делает функция СРЗНАЧ?

Функция СРЗНАЧ — это статистическая функция, которая вычисляет среднее арифметическое значение чисел в указанном диапазоне ячеек.

Среднее арифметическое — это сумма всех чисел, делённая на их количество. Функция СРЗНАЧ игнорирует текстовые значения, логические значения (ИСТИНА/ЛОЖЬ) и пустые ячейки.

📝 Синтаксис функции СРЗНАЧ

=СРЗНАЧ(число1; [число2]; ...)
число1 — обязательный аргумент, первое число или диапазон для вычисления среднего
[число2]; ... — необязательные аргументы, дополнительные числа или диапазоны

🔍 Аргументы функции

Тип аргумента Пример Результат
Отдельные числа =СРЗНАЧ(10; 20; 30) (10+20+30)/3 = 20
Диапазон ячеек =СРЗНАЧ(A1:A5) Среднее всех чисел в ячейках A1-A5
Несколько диапазонов =СРЗНАЧ(A1:A3; B1:B3) Среднее чисел в A1-A3 и B1-B3
Смешанные аргументы =СРЗНАЧ(A1:A3; 100) Среднее чисел в A1-A3 и числа 100

🧮 Примеры использования

Пример 1: Средняя оценка

Условие: В ячейках A1:A5 находятся оценки ученика по пяти предметам. Нужно вычислить средний балл.

Данные:
A1 = 4, A2 = 5, A3 = 3, A4 = 4, A5 = 5

Формула: =СРЗНАЧ(A1:A5)

Результат: (4+5+3+4+5)/5 = 21/5 = 4.2

Пример 2: Средняя температура

Условие: В ячейках B1:B7 находятся температуры за неделю. Нужно вычислить среднюю температуру.

Данные:
B1=15, B2=16, B3=14, B4=18, B5=20, B6=17, B7=19

Формула: =СРЗНАЧ(B1:B7)

Результат: (15+16+14+18+20+17+19)/7 = 119/7 = 17

Пример 3: Среднее с игнорированием текста

Условие: В ячейках C1:C5 находятся данные, включая текстовые значения. Нужно вычислить среднее только числовых значений.

Данные:
C1=10, C2="нет данных", C3=20, C4=30, C5=пусто

Формула: =СРЗНАЧ(C1:C5)

Результат: (10+20+30)/3 = 60/3 = 20
Текст "нет данных" и пустая ячейка игнорируются.

🔗 Особенности работы функции СРЗНАЧ

Игнорирование нечисловых значений

Пример: =СРЗНАЧ(A1:A4), где A1=10, A2="текст", A3=20, A4=пусто
Результат: (10+20)/2 = 15 (текст и пустая ячейка игнорируются)

Среднее с нулями

Пример: =СРЗНАЧ(A1:A4), где A1=10, A2=0, A3=20, A4=0
Результат: (10+0+20+0)/4 = 30/4 = 7.5 (нули учитываются как числа)

Среднее с логическими значениями

Пример: =СРЗНАЧ(A1:A3), где A1=10, A2=ИСТИНА, A3=20
Результат: (10+20)/2 = 15 (ИСТИНА игнорируется)

⚠️ Частые ошибки

  • Путаница с функцией СУММ — СРЗНАЧ вычисляет среднее, а СУММ складывает числа
  • Ожидание учёта текста — функция СРЗНАЧ игнорирует текстовые значения
  • Неправильный расчёт при пустых ячейках — пустые ячейки не учитываются в расчёте среднего
  • Забыть знак равенства — все формулы в Excel начинаются с =
  • Неправильные разделители — в русской версии Excel аргументы разделяются точкой с запятой (;), а не запятой (,)

🎯 Алгоритм работы с функцией СРЗНАЧ

  1. 📝 Определите, какие числа нужно усреднить
  2. 🔢 Выберите диапазон ячеек или отдельные числа
  3. 🧮 Запишите формулу: =СРЗНАЧ(диапазон_или_числа)
  4. ✅ Проверьте, что все нужные ячейки включены в диапазон
  5. 🔍 Убедитесь, что в диапазоне только числовые данные
  6. 📊 Протестируйте формулу на тестовых данных

✏️ Практическое задание

Задача: В таблице Excel записаны продажи за 6 дней: понедельник-суббота (B2:B7). В воскресенье (B8) магазин не работал. Напишите формулу для ячейки B9, которая вычисляет средние продажи за рабочие дни (игнорируя воскресенье).

Решение:
Поскольку воскресенье пустое или содержит 0, функция СРЗНАЧ автоматически его игнорирует:
=СРЗНАЧ(B2:B7)

Проверка:
Если B2=100, B3=150, B4=200, B5=180, B6=220, B7=300, B8=0
Результат: (100+150+200+180+220+300)/6 = 1150/6 ≈ 191.67

🔗 Связанные функции

СРЗНАЧЕСЛИ — вычисление среднего по одному условию
СРЗНАЧЕСЛИМН — вычисление среднего по нескольким условиям
СУММ — суммирование чисел в диапазоне
СЧЁТ — подсчёт количества чисел в диапазоне
МИН — нахождение минимального значения
МАКС — нахождение максимального значения

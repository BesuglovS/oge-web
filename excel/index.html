📊 Теория: Microsoft Excel для ОГЭ

💡 Что такое Microsoft Excel?

Microsoft Excel — это программа для работы с электронными таблицами, которая входит в пакет Microsoft Office. Excel позволяет организовывать данные в таблицы, выполнять вычисления, анализировать информацию и создавать диаграммы.

В задачах ОГЭ по информатике Excel используется для решения задач на анализ данных, вычисления с использованием формул и функций.

📝 Основные понятия Excel

🔢 Ячейка

Ячейка — основной элемент таблицы. Каждая ячейка имеет уникальный адрес, состоящий из буквы столбца и номера строки (например, A1, B5, C10).

📊 Столбец и строка

Столбцы обозначаются буквами латинского алфавита (A, B, C, ...). Строки обозначаются числами (1, 2, 3, ...).

🧮 Диапазон ячеек

Диапазон ячеек — группа смежных ячеек. Обозначается через двоеточие между адресами первой и последней ячеек (например, A1:A10, B5:D10).

📈 Формулы

Формула — выражение для вычисления значения. Все формулы в Excel начинаются со знака равенства (=). Например: =A1+B1, =СУММ(A1:A10).

📍 Абсолютная и относительная адресация

Адресация — способ указания ссылок на ячейки в формулах. В Excel существуют относительные (A1), абсолютные ($A$1) и смешанные ($A1, A$1) ссылки. Подробнее об адресации →

🧮 Основные правила работы с формулами

=НАЗВАНИЕ_ФУНКЦИИ(аргумент1; аргумент2; ...)
Общий синтаксис функций в Excel
Аргументы разделяются точкой с запятой (;)

📝 Правила записи формул

  1. Все формулы начинаются со знака равенства (=)
  2. Имена функций пишутся заглавными буквами
  3. Аргументы функций разделяются точкой с запятой (;)
  4. Текст в аргументах заключается в кавычки ("")
  5. Адреса ячеек указываются без кавычек

💡 Полезные советы

  • Для указания диапазона используйте двоеточие (A1:A10 — все ячейки от A1 до A10)
  • Для указания несмежных ячеек используйте точку с запятой (A1; B5; C10)
  • В условиях сравнения используйте операторы: = (равно), <> (не равно), > (больше), < (меньше), >= (больше или равно), <= (меньше или равно)
  • Для подсчёта всех ячеек (включая пустые и с текстом) используйте функцию СЧЁТЗ

✏️ Пример задачи с Excel

Пример: Анализ таблицы с оценками

Условие: В таблице Excel записаны оценки учеников по трём предметам. В столбце D нужно вычислить средний балл для каждого ученика. Какая формула должна быть записана в ячейке D2 и скопирована вниз?

Решение:
Данные: A — фамилия, B — математика, C — информатика, D — средний балл
Формула в D2: =СРЗНАЧ(B2:C2)
После копирования в D3: =СРЗНАЧ(B3:C3)
После копирования в D4: =СРЗНАЧ(B4:C4)
Ответ: =СРЗНАЧ(B2:C2)

⚠️ Частые ошибки

  • Забыть знак равенства — все формулы в Excel начинаются с =
  • Неправильные разделители — в русской версии Excel аргументы разделяются точкой с запятой (;), а не запятой (,)
  • Путаница в относительных и абсолютных ссылках — $A$1 — абсолютная ссылка, A1 — относительная
  • Неправильные операторы сравнения — <> значит "не равно", а не "больше или меньше"
  • Забыть кавычки для текста — текстовые значения в условиях должны быть в кавычках: ="отлично"

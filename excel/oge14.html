📊 Задание 14 ОГЭ — Электронные таблицы

💡 Что такое задание 14?

Задание 14 — практическое задание ОГЭ по информатике, в котором требуется выполнить обработку данных в электронных таблицах (Microsoft Excel или LibreOffice Calc).

В задании дана таблица с исходными данными. Необходимо:

  • Выполнить вычисления с использованием формул и функций
  • Построить круговую диаграмму
  • Сохранить полученную таблицу

В отличие от задания 13.2 (текстовый редактор), задание 14 проверяет умение работать с формулами, использовать встроенные функции и строить диаграммы.

📋 Пример задания 14 — Вариант 1 (Грузоперевозки)

В электронную таблицу внесли информацию о грузоперевозках, совершённых некоторым автопредприятием с 1 по 9 октября. Ниже приведены первые пять строк таблицы.

A B C D E F
Дата Пункт отправления Пункт назначения Расстояние Расход бензина Масса груза
1 октября Липки Берёзки 432 63 770
1 октября Орехово Дубки 121 17 670
1 октября Осинки Вязово 333 47 830
1 октября Липки Вязово 384 54 730

Всего в таблицу внесены данные о 370 перевозках.

Задания к таблице

  1. Какова суммарная протяжённость всех перевозок, произведённых с 7 по 9 октября? Ответ запишите в ячейку H2 таблицы.
  2. Какова средняя масса груза при автоперевозках из города Осинки? Ответ запишите в ячейку H3 таблицы с точностью не менее одного знака после запятой.
  3. Постройте круговую диаграмму, отображающую соотношение количества перевозок 1 октября, 2 октября и 3 октября. Левый верхний угол диаграммы разместите вблизи ячейки G6. В поле диаграммы должны присутствовать легенда и числовые значения данных.

📋 Пример задания 14 — Вариант 2 (Тестирование учеников)

В электронную таблицу занесли данные о тестировании учеников по выбранным ими предметам.

A B C D
округ фамилия предмет балл
С Ученик 1 физика 240
В Ученик 2 физкультура 782
Ю Ученик 3 биология 361
СВ Ученик 4 обществознание 377

Всего в таблицу внесены данные 1000 учеников.

Задания к таблице

  1. Определите, сколько учеников, которые проходили тестирование по биологии, набрали более 500 баллов. Ответ запишите в ячейку H2 таблицы.
  2. Вычислите средний тестовый балл у учеников, которые проходили тестирование по биологии. Ответ запишите в ячейку H3 таблицы с точностью не менее двух знаков после запятой.
  3. Постройте круговую диаграмму, отображающую соотношение числа участников из округов с кодами «В», «Зел» и «З». Левый верхний угол диаграммы разместите вблизи ячейки G6. В поле диаграммы должны присутствовать легенда и числовые значения данных.

🧮 Необходимые функции Excel для задания 14

Для решения задания 14 потребуются следующие функции. Нажмите на название функции, чтобы перейти к подробному описанию:

📋 ЕСЛИ (IF)

Проверяет условие и возвращает одно значение, если оно ИСТИНА, и другое, если ЛОЖЬ.

Применение: для фильтрации данных по условию (например, =ЕСЛИ(C2="биология"; 1; 0))

🔗 И (AND)

Возвращает ИСТИНА, если все условия истинны.

Применение: для проверки нескольких условий одновременно (например, =ЕСЛИ(И(C2="биология"; D2>500); 1; 0))

➕ СУММ (SUM)

Суммирует числа в диапазоне ячеек.

Применение: для подсчёта суммы значений (например, =СУММ(E2:E1001))

🔍 СЧЁТЕСЛИ (COUNTIF)

Подсчитывает количество ячеек, удовлетворяющих условию.

Применение: для подсчёта количества записей по условию (например, =СЧЁТЕСЛИ(A2:A1001; "В"))

➕🔍 СУММЕСЛИ (SUMIF)

Суммирует ячейки по заданному условию.

Применение: для суммирования значений, соответствующих критерию

📊 СРЗНАЧ (AVERAGE)

Вычисляет среднее арифметическое чисел в диапазоне.

Применение: для расчёта среднего значения (например, средний балл)

📊🔍 СРЗНАЧЕСЛИ (AVERAGEIF)

Вычисляет среднее арифметическое по заданному условию.

Применение: для расчёта среднего только для определённых записей

🛠️ Пошаговый алгоритм решения задания 14

1
Откройте файл с электронной таблицей, который выдадут организаторы экзамена. Внимательно изучите структуру данных: какие столбцы есть, какие данные в них содержатся.
2
Прочитайте условие задачи. Определите, какие вычисления нужно сделать и какие функции для этого потребуются. Обратите внимание на точность ответа (сколько знаков после запятой требуется).
3
Добавьте вспомогательные столбцы при необходимости. Например, для подсчёта количества записей, удовлетворяющих условию, используйте функцию =ЕСЛИ(условие; 1; 0).
4
Запишите формулы.
  • Для суммы: =СУММ(диапазон)
  • Для среднего: =СРЗНАЧ(диапазон) или =СУММ(диапазон)/СЧЁТ(диапазон)
  • Для подсчёта по условию: =СЧЁТЕСЛИ(диапазон; критерий)
5
Запишите ответы в указанные ячейки (обычно H2, H3). Убедитесь, что формат ячейки отображает нужное количество знаков после запятой.
6
Постройте круговую диаграмму.
  • Выделите данные, которые нужно отобразить на диаграмме (категории и их числовые значения)
  • Вставка → Диаграмма → Круговая
  • Добавьте легенду и подписи данных с числовыми значениями
  • Разместите диаграмму вблизи указанной ячейки (обычно G6)
7
Проверьте результат. Убедитесь, что все ответы записаны в правильные ячейки, диаграмма содержит легенду и числовые значения данных. Сохраните файл под именем, указанным организаторами.

🎯 Ключевые функции и их применение

Функция ЕСЛИ с логическими операциями

=ЕСЛИ(И(C2="биология"; D2>500); 1; 0)
Проверяет, что в ячейке C2 написано "биология" И в D2 значение больше 500.
Если оба условия верны — возвращает 1, иначе — 0.
Затем СУММ по столбцу даст количество учеников, сдавших биологию на >500 баллов.

Функция СЧЁТЕСЛИ для подсчёта по условию

=СЧЁТЕСЛИ(A2:A1001; "В")
Подсчитывает количество ячеек в диапазоне A2:A1001,
которые содержат значение "В".
Используется для подсчёта количества записей, соответствующих критерию.

Вычисление среднего с использованием вспомогательных столбцов

=СУММ(F2:F1001) / СЧЁТЕСЛИ(F2:F1001; ">0")
Сумма всех значений в столбце F (где предварительно записаны баллы только по биологии)
делится на количество ненулевых записей (т.е. количество учеников по биологии).
Результат — средний балл по биологии.

Построение круговой диаграммы

Подготовка данных → Вставка → Диаграмма → Круговая
Для построения диаграммы необходимо сначала подготовить данные:
1. Записать названия категорий (например, названия округов)
2. Вычислить количество записей для каждой категории с помощью СЧЁТЕСЛИ
3. Выделить подготовленные данные → Вставить круговую диаграмму
4. Добавить легенду и подписи данных (числовые значения)

⚠️ Типичные ошибки и как их избежать

❌ Неправильный синтаксис формул

Ошибка: использование запятой вместо точки с запятой для разделения аргументов функции.

Решение: в русской версии Excel и LibreOffice Calc аргументы разделяются точкой с запятой (;), а не запятой (,).

❌ Ошибки в текстовых критериях

Ошибка: несоответствие регистра или написание текста в критерии (например, "Биология" вместо "биология").

Решение: внимательно сверяйте написание текстовых значений в условии. В Excel текстовые значения чувствительны к регистру в функциях ЕСЛИ, но в СЧЁТЕСЛИ регистр обычно не учитывается.

❌ Неправильный диапазон данных

Ошибка: неверно указан диапазон ячеек (например, A2:A1000 вместо A2:A1001).

Решение: проверьте, сколько строк данных в таблице. Если данных 1000 строк (строки 2–1001), то диапазон должен быть A2:A1001.

❌ Неправильная точность ответа

Ошибка: ответ записан с недостаточным количеством знаков после запятой.

Решение: настройте формат ячейки (Формат → Ячейки → Число → Числовой) с нужным количеством десятичных знаков. Если в задании сказано «не менее одного знака» — покажите минимум 1 знак.

❌ Отсутствие легенды или подписей на диаграмме

Ошибка: на круговой диаграмме нет легенды или не указаны числовые значения данных.

Решение: после создания диаграммы добавьте легенду (через контекстное меню или Работа с диаграммами) и подписи данных, выбрав отображение «Значения».

❌ Диаграмма не в том месте

Ошибка: диаграмма размещена не вблизи указанной ячейки.

Решение: перетащите диаграмму так, чтобы её левый верхний угол находился возле указанной ячейки (обычно G6).

🔑 Ключевые правила для успешного выполнения

  • Внимательно читайте условие. Определите, какие именно данные нужно подсчитать и с какой точностью.
  • Используйте вспомогательные столбцы. Если нужно отфильтровать данные по условию, создайте дополнительный столбец с функцией ЕСЛИ.
  • Проверяйте диапазоны. Убедитесь, что формулы охватывают все строки таблицы.
  • Настройте формат ячеек с ответами — числовой формат с нужным количеством десятичных знаков.
  • Не забывайте про диаграмму. Добавьте легенду, подписи данных (числовые значения), разместите в указанном месте.
  • Проверьте ответы. Убедитесь, что числовые значения записаны в правильные ячейки (H2, H3).
  • Сохраните файл под именем, указанным организаторами экзамена.

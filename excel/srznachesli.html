📊 Функция СРЗНАЧЕСЛИ в Excel

💡 Что делает функция СРЗНАЧЕСЛИ?

Функция СРЗНАЧЕСЛИ — это статистическая функция, которая вычисляет среднее арифметическое значений ячеек в диапазоне, удовлетворяющих заданному условию.

Функция СРЗНАЧЕСЛИ используется для анализа данных, когда нужно вычислить среднее значение только для тех элементов, которые соответствуют определённому условию.

📝 Синтаксис функции СРЗНАЧЕСЛИ

=СРЗНАЧЕСЛИ(диапазон; условие; [диапазон_усреднения])
диапазон — диапазон для проверки условия
условие — критерий для отбора ячеек
диапазон_усреднения — необязательный, значения для усреднения

🧮 Примеры использования

Пример: Средний балл учеников с оценкой "5"

Условие: В столбце A оценки, в столбце B баллы. Вычислить средний балл учеников с оценкой "5".

Данные:
A1:A5 = {5, 4, 5, 3, 5}
B1:B5 = {95, 80, 90, 70, 85}

Формула: =СРЗНАЧЕСЛИ(A1:A5; 5; B1:B5)

Расчёт:
Ученики с оценкой 5: A1, A3, A5
Их баллы: 95, 90, 85
Среднее: (95 + 90 + 85) / 3 = 90
Результат: 90

🔗 Связанные функции

СРЗНАЧ — вычисляет среднее арифметическое всех значений
СРЗНАЧЕСЛИМН — вычисляет среднее по нескольким условиям
СЧЁТЕСЛИ — подсчитывает количество ячеек по условию
СУММЕСЛИ — суммирует значения по условию

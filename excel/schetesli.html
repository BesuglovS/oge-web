🔢 Функция СЧЁТЕСЛИ в Excel

💡 Что делает функция СЧЁТЕСЛИ?

Функция СЧЁТЕСЛИ — это статистическая функция, которая подсчитывает количество ячеек в диапазоне, удовлетворяющих заданному условию. Функция возвращает целое число — количество ячеек, которые соответствуют критерию.

Функция СЧЁТЕСЛИ используется для анализа данных, когда нужно узнать, сколько элементов в списке соответствуют определённому условию (например, сколько учеников получили оценку "5", сколько товаров стоят больше 1000 рублей и т.д.).

📝 Синтаксис функции СЧЁТЕСЛИ

=СЧЁТЕСЛИ(диапазон; критерий)
диапазон — обязательный аргумент, диапазон ячеек для проверки
критерий — обязательный аргумент, условие для подсчёта ячеек

🔍 Аргументы функции

Аргумент Описание Пример
диапазон Диапазон ячеек, в которых нужно подсчитать ячейки по условию A1:A10, B2:B100
критерий Условие для подсчёта (число, текст, выражение сравнения) 5, "отлично", ">100"

📊 Типы критериев в функции СЧЁТЕСЛИ

Тип критерия Пример Что подсчитывает
Числовое значение =СЧЁТЕСЛИ(A1:A10; 5) Количество ячеек, равных 5
Текстовое значение =СЧЁТЕСЛИ(B1:B10; "отлично") Количество ячеек с текстом "отлично"
Выражение сравнения =СЧЁТЕСЛИ(C1:C10; ">100") Количество ячеек больше 100
Символы подстановки =СЧЁТЕСЛИ(D1:D10; "А*") Количество ячеек, начинающихся с "А"
Ссылка на ячейку =СЧЁТЕСЛИ(E1:E10; F1) Количество ячеек, равных значению в F1

🧮 Примеры использования

Пример 1: Подсчёт оценок "5"

Условие: В диапазоне A1:A20 находятся оценки учеников. Нужно подсчитать, сколько учеников получили оценку "5".

Данные: A1:A20 = {4, 5, 3, 5, 4, 5, 2, 5, 4, 3, 5, 4, 5, 3, 4, 5, 2, 5, 4, 5}

Формула: =СЧЁТЕСЛИ(A1:A20; 5)

Расчёт:
Подсчитываем ячейки со значением 5: позиции 2, 4, 6, 8, 11, 13, 16, 18, 20
Всего: 9 ячеек
Результат: 9

Пример 2: Подсчёт товаров дороже 1000 рублей

Условие: В диапазоне B1:B50 находятся цены товаров. Нужно подсчитать, сколько товаров стоят больше 1000 рублей.

Данные: B1:B50 (предположим, 15 товаров > 1000)

Формула: =СЧЁТЕСЛИ(B1:B50; ">1000")

Расчёт:
Проверяем каждую ячейку: если цена > 1000, считаем
Результат: 15

Пример 3: Подсчёт фамилий на букву "И"

Условие: В диапазоне C1:C100 находятся фамилии учеников. Нужно подсчитать, сколько фамилий начинаются на букву "И".

Данные: C1:C100 (предположим, 8 фамилий на "И")

Формула: =СЧЁТЕСЛИ(C1:C100; "И*")

Расчёт:
* — символ подстановки, означает "любые символы после"
"И*" — текст, начинающийся с "И"
Результат: 8

🔗 Особенности работы функции СЧЁТЕСЛИ

Символы подстановки

* — заменяет любое количество символов
? — заменяет один символ
~ — экранирует символы * и ?
Пример: =СЧЁТЕСЛИ(A1:A10; "Ив*") — фамилии, начинающиеся с "Ив"

Регистр букв

Правило: Функция СЧЁТЕСЛИ не учитывает регистр букв.
Пример: =СЧЁТЕСЛИ(A1:A10; "отлично") подсчитает и "Отлично", и "ОТЛИЧНО", и "отлично"

Пустые и непустые ячейки

Пустые ячейки: =СЧЁТЕСЛИ(A1:A10; "")
Непустые ячейки: =СЧЁТЕСЛИ(A1:A10; "<>")
Пример: =СЧЁТЕСЛИ(B1:B100; "<>") — подсчёт заполненных ячеек

⚠️ Частые ошибки

  • Забывание кавычек для текста — =СЧЁТЕСЛИ(A1:A10; отлично) неправильно, нужно =СЧЁТЕСЛИ(A1:A10; "отлично")
  • Неправильные кавычки для выражений — =СЧЁТЕСЛИ(A1:A10; >100) неправильно, нужно =СЧЁТЕСЛИ(A1:A10; ">100")
  • Путаница с диапазонами — =СЧЁТЕСЛИ(A1; ">5") неправильно (одиночная ячейка), нужно =СЧЁТЕСЛИ(A1:A10; ">5")
  • Использование запятой вместо точки с запятой — в русской версии Excel аргументы разделяются точкой с запятой (;)
  • Забывание символа * для частичного совпадения — =СЧЁТЕСЛИ(A1:A10; "Ив") не найдёт "Иванов", нужно =СЧЁТЕСЛИ(A1:A10; "Ив*")

🎯 Алгоритм работы с функцией СЧЁТЕСЛИ

  1. 📝 Определите диапазон ячеек для анализа
  2. 🔍 Сформулируйте условие для подсчёта
  3. 📋 Запишите условие в правильном формате (число, текст в кавычках, выражение в кавычках)
  4. 🧮 Запишите формулу: =СЧЁТЕСЛИ(диапазон; критерий)
  5. ✅ Проверьте, что критерий правильно записан (кавычки, символы сравнения)
  6. 📊 Протестируйте формулу на примере данных

✏️ Практическое задание

Задача: В таблице Excel записаны данные о результатах экзамена: столбец A — фамилии учеников, столбец B — баллы (от 0 до 100). Напишите формулу для ячейки C1, которая подсчитывает количество учеников, набравших не менее 60 баллов (проходной балл).

Решение:
1. Диапазон для анализа: B1:B100 (баллы учеников)
2. Условие: балл ≥ 60
3. Формулировка условия: ">=60"
4. Формула: =СЧЁТЕСЛИ(B1:B100; ">=60")

Проверка:
• Если в B1:B100 есть значения: 45, 78, 92, 55, 61, 59, 85, 72
• Условию >=60 удовлетворяют: 78, 92, 61, 85, 72 (5 значений)
• Результат формулы: 5

🔗 Связанные функции

СЧЁТ — подсчитывает количество числовых значений в диапазоне
СЧЁТЕСЛИМН — подсчитывает количество ячеек, удовлетворяющих нескольким условиям
СУММЕСЛИ — суммирует значения ячеек, удовлетворяющих условию
СУММЕСЛИМН — суммирует значения ячеек, удовлетворяющих нескольким условиям
СРЗНАЧЕСЛИ — вычисляет среднее значение ячеек, удовлетворяющих условию

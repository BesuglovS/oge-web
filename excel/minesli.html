📉 Функция МИНЕСЛИ в Excel

💡 Что делает функция МИНЕСЛИ?

Функция МИНЕСЛИ — это статистическая функция, которая находит минимальное значение среди ячеек, удовлетворяющих заданному условию.

Функция МИНЕСЛИ используется для анализа данных, когда нужно найти наименьшее значение только среди элементов, соответствующих определённому условию.

📝 Синтаксис функции МИНЕСЛИ

=МИНЕСЛИ(диапазон; условие; [диапазон_поиска])
диапазон — диапазон для проверки условия
условие — критерий для отбора ячеек
диапазон_поиска — необязательный, значения для поиска минимума

🧮 Пример использования

Пример: Минимальная зарплата в отделе "IT"

Условие: В столбце A отделы, в столбце B зарплаты. Найти минимальную зарплату в отделе "IT".

Данные:
A1:A6 = {"IT", "HR", "IT", "Sales", "IT", "HR"}
B1:B6 = {50000, 40000, 55000, 45000, 48000, 42000}

Формула: =МИНЕСЛИ(A1:A6; "IT"; B1:B6)

Расчёт:
Сотрудники отдела IT: A1, A3, A5
Их зарплаты: 50000, 55000, 48000
Минимальная: 48000
Результат: 48000

🔗 Связанные функции

МИН — находит минимальное значение в диапазоне
МАКСЕСЛИ — находит максимальное значение по условию
СЧЁТЕСЛИ — подсчитывает количество ячеек по условию
СУММЕСЛИ — суммирует значения по условию

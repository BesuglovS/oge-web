🔍 Задание 4 — Поиск кратчайшего пути в графе

⚙️ Настройка матрицы смежности

По умолчанию буквы английского алфавита A, B, C, …

📊 Матрица расстояний

Введите расстояния между вершинами. Пустая ячейка означает отсутствие прямой дороги. Матрица симметрична — заполняйте только верхнюю половину (над диагональю).

⚡ Алгоритм Дейкстры Для поиска кратчайшего пути используется алгоритм Дейкстры — классический алгоритм на графах, который находит кратчайшие пути от одной вершины до всех остальных. Сложность алгоритма: O(N²), где N — количество вершин.
